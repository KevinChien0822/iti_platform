# 啟動專案前的準備接手文件

## 專案目標
建立 ITI 校友福利社平台，能夠：
1. 發布校友招人訊息（串接人力銀行或超連結）
2. 發布討論帖子
3. 未來社團進駐
4. 上架商品或串接官網商品
5. 發布優惠活動
6. 身分驗證流程
7. 預留擴充空間

## 可行性判斷
✅ 可行。標準 CRUD + 驗證流程，技術棧成熟，單人可完成基礎框架。

## 資源評估
- 前端：Next.js 16 + React 19 + Tailwind CSS v4
- 後端：Next.js API Routes
- 資料庫：Prisma 7 + SQLite（未來可升級 PostgreSQL）
- 驗證：NextAuth v5（支援 Email/密碼 + OAuth）
- 部署：Vercel / 自架
- 時間：基礎框架約 2-4 小時

## 風險評估
- 🔴 身分驗證流程需確保校友身份過濾 → 用管理員審核機制
- 🟡 資料庫設計需預留社團、商品、優惠的擴充欄位
- 🟢 單頁應用基礎功能明確，技術包袱小
