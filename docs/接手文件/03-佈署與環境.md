### PostgreSQL (VPS)

| Port | 5433 |
| 資料庫名稱 | iti_alumni_db |
| 使用者 | iti_app |
| 密碼 | iti_alumni_2024 |
| 版本 | PostgreSQL 16 (Alpine) |
| 容器名稱 | iti-postgres |

### Docker 指令

```bash
# 連線到資料庫
docker exec -it iti-postgres psql -U iti_app -d iti_alumni_db

# 備份資料庫
docker exec iti-postgres pg_dump -U iti_app iti_alumni_db > backup.sql

# 還原資料庫
cat backup.sql | docker exec -i iti-postgres psql -U iti_app -d iti_alumni_db

# 重新啟動容器
docker restart iti-postgres

# 查看容器日誌
docker logs iti-postgres
```

### VPS 防火牆

UFW 已開放 Port 5433（僅來自 Vercel 的連線可視需求收緊）。

## 本地開發環境

```env
DATABASE_URL="file:./prisma/dev.db"   # SQLite
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="dev-secret-change-in-production-iti-platform"
```

本地啟動：
```bash
npm install
npx prisma generate
npx prisma db push
npx tsx prisma/seed.ts
npm run dev
```
